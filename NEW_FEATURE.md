## Voice Channel Join with Realtime Openai API

We want to implement the possibility for a user to ask the bot to join the voice channel and while in the voice channel the bot should use the openai realtime api in order to act as a voice assitant there that responds to user voice input.

You can use context7 and/or the openaiDeveloperDocs mcp tools in order to get up to date documentation on libraries like discord and openai and openai API.
